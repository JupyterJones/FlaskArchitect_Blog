Changing the cursor theme on Linux Ubuntu involves a few steps. Here’s how you can do it:

### Step 1: Install a New Cursor Theme
You can find various cursor themes online. Popular sources include GNOME Look and the Ubuntu Software Center. Here’s how to install a theme using the terminal:

1. **Download a Cursor Theme:**
   Visit a website like [GNOME Look](https://www.gnome-look.org/) and download a cursor theme you like.

2. **Extract the Cursor Theme:**
   Extract the downloaded `.tar.gz` or `.zip` file.

3. **Move the Cursor Theme to the Icons Directory:**
   Move the extracted theme folder to the appropriate directory. Open a terminal and run:
   ```bash
   sudo mv /path/to/extracted/theme /usr/share/icons/
   ```

### Step 2: Update the Cursor Theme
You need to update your system settings to use the new cursor theme.

1. **Edit the Cursor Configuration File:**
   Open the `default` cursor file in a text editor:
   ```bash
   sudo nano /usr/share/icons/default/index.theme
   ```
   
2. **Change the Cursor Theme Name:**
   Update the `Inherits` line with the name of your new cursor theme:
   ```ini
   [Icon Theme]
   Inherits=YourNewCursorTheme
   ```

3. **Update `alternatives`:**
   Update the alternatives system to use the new cursor theme:
   ```bash
   sudo update-alternatives --install /usr/share/icons/default/index.theme x-cursor-theme /usr/share/icons/default/index.theme 100
   sudo update-alternatives --set x-cursor-theme /usr/share/icons/default/index.theme
   ```

### Step 3: Apply the New Cursor Theme
To ensure that the new cursor theme is applied system-wide, you can restart your system. Alternatively, you can restart your display manager (e.g., GDM, LightDM) with the following command:
```bash
sudo systemctl restart gdm
```
or
```bash
sudo systemctl restart lightdm
```

### Step 4: Optional - Use GNOME Tweaks (For GNOME Users)
If you’re using GNOME, you can use GNOME Tweaks to change the cursor theme:

1. **Install GNOME Tweaks:**
   ```bash
   sudo apt install gnome-tweaks
   ```

2. **Open GNOME Tweaks:**
   Run `gnome-tweaks` from the terminal or find it in your applications menu.

3. **Change the Cursor Theme:**
   Go to the "Appearance" section and select your new cursor theme from the "Cursor" dropdown menu.

After following these steps, your cursor theme should be updated to the new one you selected. If you encounter any issues, make sure the cursor theme is properly installed and that the theme name matches exactly in the configuration files.
