```javascript
// Function to find and highlight the search string
function findString(str) {
    if (parseInt(navigator.appVersion) < 4) return;

    // Check if find method is supported
    if (window.find) {
        // Find the search string
        var strFound = window.find(str);
        if (!strFound) {
            // If not found, try to find from the beginning
            window.find(str, 0, 1);
        }
        if (strFound) {
            // Highlight the found text
            var range = window.getSelection().getRangeAt(0);
            var span = document.createElement("span");
            span.style.backgroundColor = "yellow";
            range.surroundContents(span);
        }
    } else if (navigator.appName.indexOf("Microsoft") != -1) {
        // Handle Microsoft browsers
        // Not implemented for brevity
    } else if (navigator.appName == "Opera") {
        // Handle Opera browsers
        alert("Opera browsers not supported, sorry...");
        return;
    }

    // If not found, show alert
    if (!strFound) alert("String '" + str + "' not found!");
}

// Function to move cursor to next occurrence of search input
function moveToNextOccurrence() {
    var search_str = document.getElementById("search_input").value;
    findString(search_str);
}
```

This is the JavaScript part of the code. It contains two functions:

1. `findString(str)`: This function takes a string `str` as input. It first checks if the browser supports the `find` method. If supported, it tries to find the string in the document. 2. `moveToNextOccurrence()`: This function is called when the button is clicked. It retrieves the value from the input field with the id "search_input" and passes it to the `findString` function to find and highlight the next occurrence of the string in the document.

Overall, this script provides a simple search functionality for finding and highlighting text in a web page. It's important to note that the script may not work in all browsers, especially older ones like Internet Explorer, and it provides basic error handling for unsupported browsers.
