To overlay a video on top of another using MoviePy and ffmpeg, and apply colorchrome effect, make black transparent, and offset the videos by 0.5 seconds, you can use the following code:

```python
from moviepy.editor import VideoFileClip, concatenate_videoclips, ColorClip
from moviepy.video.fx import colorx, mask_and
from moviepy.video.io.ffmpeg_tools import ffmpeg_extract_subclip

# Load the videos
video_bottom = VideoFileClip("video_bottom.mp4")
video_top = VideoFileClip("video_top.mp4")

# Apply colorchrome effect to the top video
video_top = colorx.colorx(video_top, color=(0, 0, 0))

# Make black transparent in the top video
mask = video_top.fx(mask_and, ColorClip(video_top.size, color=(0, 0, 0), ismask=True))

# Offset the videos by 0.5 seconds
video_bottom = video_bottom.subclip(0.5)
video_top = video_top.subclip(0, video_bottom.duration)

# Overlay the videos
result = concatenate_videoclips([video_bottom.set_position('center'), video_top.set_position('center')]).set_mask(mask)

# Export the result
result.write_videofile("output.mp4", codec="libx264")
```

Make sure to replace `"video_bottom.mp4"` and `"video_top.mp4"` with the actual filenames of your videos. The resulting video will be saved as `"output.mp4"`.
