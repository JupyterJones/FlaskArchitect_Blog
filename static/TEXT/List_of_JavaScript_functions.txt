You can find a comprehensive list of JavaScript native functions in the official JavaScript documentation provided by MDN Web Docs (Mozilla Developer Network). MDN is a reliable and detailed resource for JavaScript and web development in general.

Here are the direct links to the relevant sections on MDN:

1. **Global Objects:** This section includes built-in objects such as `Object`, `Array`, `Function`, `String`, `Number`, and more.
   - [MDN Web Docs - Global Objects](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects)

2. **Global Functions:** This section includes global functions like `eval()`, `isFinite()`, `isNaN()`, `parseFloat()`, `parseInt()`, and more.
   - [MDN Web Docs - Global Functions](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects#functions)

3. **Statements and Declarations:** This section covers all the statements like `if...else`, `switch`, `try...catch`, loops (`for`, `while`, etc.), and variable declarations (`var`, `let`, `const`).
   - [MDN Web Docs - Statements and Declarations](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements)

4. **Operators:** This section provides information about all the operators available in JavaScript, including arithmetic, comparison, logical, assignment, and more.
   - [MDN Web Docs - Operators](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators)

By exploring these sections on MDN Web Docs, you can find detailed explanations, syntax, examples, and usage guidelines for all native JavaScript functions and objects.
