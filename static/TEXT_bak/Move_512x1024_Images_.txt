You can use the following Bash script to achieve this:

```bash
#!/bin/bash

# Create the destination directory if it doesn't exist
mkdir -p 512x1024

# Iterate over all the .jpg files in the current directory
for file in *.jpg; do
    # Check the dimensions of each image using ImageMagick's identify command
    dimensions=$(identify -format "%wx%h" "$file")

    # Check if the dimensions match 512x1024
    if [ "$dimensions" = "512x1024" ]; then
        # Move the image to the 512x1024 directory
        mv "$file" 512x1024/
    fi
done
```

Here's a breakdown of how the script works:

1. The `mkdir -p 512x1024` command creates the destination directory (`512x1024`) if it doesn't exist. The `-p` flag ensures that the command doesn't produce an error if the directory already exists.

2. The `for` loop iterates over all the `.jpg` files in the current directory.

3. The `identify -format "%wx%h" "$file"` command uses ImageMagick's `identify` tool to retrieve the dimensions of each image file. The `%wx%h` format specifier extracts the width and height in pixels.

4. The script compares the obtained dimensions (`$dimensions`) with the desired dimensions (`512x1024`).

5. If the dimensions match, the script moves the image file to the `512x1024` directory using the `mv` command.

Make sure you have ImageMagick installed on your system for the `identify` command to work correctly.
