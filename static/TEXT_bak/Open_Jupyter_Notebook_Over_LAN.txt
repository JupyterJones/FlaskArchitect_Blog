To open a Jupyter Notebook over a local area network (LAN), you can follow these steps:

1. **Install Jupyter Notebook**: If you haven't already, you need to install Jupyter Notebook. You can do this using pip:

   ```
   pip install jupyter
   ```

2. **Start a Jupyter Notebook Server**: Open your terminal or command prompt, navigate to the directory where you want to work, and run the following command:

   ```
   jupyter notebook
   ```

   This will start the Jupyter Notebook server locally, and it will provide you with a URL like `http://localhost:8888/` in your terminal.

3. **Access Jupyter Notebook Over LAN**:
   To access your Jupyter Notebook over the LAN, you need to find your computer's local IP address. You can usually find this by running the following command:

   - On Linux or macOS:

     ```
     ifconfig
     ```

   - On Windows:

     ```
     ipconfig
     ```

   Look for the IPv4 address of your active network connection (usually labeled as "inet" or "IPv4 Address").

4. Using a device on your LAN (e.g., another computer or a smartphone), open a web browser and enter the following address:

   ```
   http://your_local_ip:8888
   ```

   Replace `your_local_ip` with the actual IP address you found in step 3. Also, ensure that the port (default is 8888) matches the one displayed when you started the Jupyter Notebook server.

5. You should now have access to your Jupyter Notebook over the LAN, and you can create, open, and edit notebooks as usual.

Please note that you might need to adjust your firewall or network settings to allow access over the LAN. Also, make sure your devices are connected to the same network for this to work.
